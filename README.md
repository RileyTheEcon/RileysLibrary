# Riley's Library

This library is a repository of the random bits of code that I've developed, found useful and wanted to hold onto. Included in this library is a number of decorators for caching, scheduling and retrying tasks. There are also class objects for logging console outputs and webscraping. This is a constantly growing and evolving library, so check back regularly.


## How to install
```bash
:: Run in Anaconda CLI with project env active
pip install git+https://github.com/RileyTheEcon/RileysLibrary.git
```

## How to update
```bash
:: Run in Anaconda CLI with project env active
pip install --upgrade --force-reinstall --no-deps git+https://github.com/RileyTheEcon/RileyLibrary.git
```
